== Description ==
On Sale is a clean and crisp child theme for the WooCommerce reference theme Storefront. You must install Storefront before using this theme.

On Sale has a simple color palette which can be edited in the customizer with no coding required. You also have the ability to set your logo, favicon and page layout options in the customizer meaning you can easily edit the theme to suit your brand look and feel. The home page product slider is great way to feature your latest and great products.
